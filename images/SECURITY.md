# Security Policy

## Supported Versions
This project is currently under active development. Only the latest version is supported.

## Reporting a Vulnerability
If you discover a security vulnerability, please do not open a public issue.

Instead, report it by:
- Sending a private message to the repository owner
- Or using GitHub Security Advisories

Please include:
- Description of the issue
- Steps to reproduce
- Possible impact

We appreciate responsible disclosure and will address issues as soon as possible.
